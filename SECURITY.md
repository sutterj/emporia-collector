# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Do not open a public issue.**

Instead, email [jesutter23@me.com](mailto:jesutter23@me.com) with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

I will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| latest  | ✅        |

## Scope

This project handles Emporia cloud credentials (email/password) and MQTT broker credentials.  Security issues related to credential handling, authentication, or network exposure are in scope.
